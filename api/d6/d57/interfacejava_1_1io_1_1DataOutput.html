Detailed Description

Defines an interface for classes that are able to write big-endian typed data to some target.

Typically, this data can be read in by a class which implements DataInput. Types that can be written include byte, 16-bit short, 32-bit int, 32-bit float, 64-bit long, 64-bit double, byte strings, and {ref DataInput MUTF-8} encoded strings.
