Detailed Description

Wraps an existing {ref OutputStream} and provides convenience methods for writing common data types in a human readable format.

This is not to be confused with DataOutputStream which is used for encoding common data types so that they can be read back in. No

IOException

is thrown by this class. Instead, callers should use {ref #checkError()} to see if a problem has occurred in this stream.
