int java.lang.Double.hashCode ( )

Returns a hashcode for this Double object.

The result is the exclusive OR of the two halves of the long integer bit representation, exactly as produced by the method {ref doubleToLongBits(double)}, of the primitive double value represented by this Double object. That is, the hashcode is the value of the expression: 

(int)(v^(v>>>32))

where v is defined by: 

long v = Double.doubleToLongBits(this.doubleValue());
Returns
a hash code value for this object.
