If the jar URL doesn't start with schema, or if it is not an absolute path, make out it with jad URL.

Such as the following jar URL, which should be made out: foo.jar foo/foo.jar But the following should not: http://foo/foo.jar file:///foo/foo.jar /foo/foo.jar
