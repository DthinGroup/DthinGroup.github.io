Detailed Description

A base class for.

Map

implementations.

Subclasses that permit new mappings to be added must override {ref put}.

The default implementations of many methods are inefficient for large maps. For example in the default implementation, each call to {ref get} performs a linear iteration of the entry set. Subclasses should override such methods to improve their performance.

1.2
