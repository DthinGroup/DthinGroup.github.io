Detailed Description

Identifies fields in formatted strings.

If a

FieldPosition

is passed to the format method with such a parameter, then the indices will be set to the start and end indices of the field in the formatted string.

A

FieldPosition

can be created by using the integer constants in the various format classes (for example

NumberFormat.INTEGER_FIELD

) or one of the fields of type

Format.Field

.

If more than one field information is needed, the method {ref NumberFormat::formatToCharacterIterator(Object)} should be used.
