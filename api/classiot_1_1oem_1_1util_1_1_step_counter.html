Detailed Description

This class is used for calculating steps through acceleration data, which supports both two axis accelerometer and three axis accelerometer. According to the constructor of StepCounter, there are below tow ways of using StepCounter class: 1) StepCounter counter = new StepCounter(xInAc, yInAc, zInAc); counter.getStepCount(); //get result. 2) StepCounter counter = new StepCounter(); counter.caculateSteps(xInAc, yInAc, zInAc); //get result.
