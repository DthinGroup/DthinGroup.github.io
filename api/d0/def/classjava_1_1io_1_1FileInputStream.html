Detailed Description

An input stream that reads bytes from a file. 

File file = ...


InputStream in = null;


try {


  in = new BufferedInputStream(new FileInputStream(file));


  ...


} finally {


  if (in != null) {


    in.close();


  }


}

This stream is not buffered. Most callers should wrap this stream with a {ref BufferedInputStream}.

Use {ref FileReader} to read characters, as opposed to bytes, from a file.
