java.lang.Cloneable Interface Reference

This (empty) interface must be implemented by all classes that wish to support cloning. More...

Detailed Description

This (empty) interface must be implemented by all classes that wish to support cloning.

The implementation of

clone()

in

Object

checks if the object being cloned implements this interface and throws

CloneNotSupportedException

if it does not.
