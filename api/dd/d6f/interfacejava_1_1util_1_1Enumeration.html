Detailed Description

A legacy iteration interface.

New code should use {ref Iterator} instead.

Iterator

replaces the enumeration interface and adds a way to remove elements from a collection.

If you have an

Enumeration

and want a

Collection

, you can use {ref Collections::list} to get a

List

If you need an

Enumeration

for a legacy API and have a

Collection
