Detailed Description

A modifiable {ref CharSequence sequence of characters} for use in creating strings.

This class is intended as a direct replacement of {ref StringBuffer} for non-concurrent use; unlike

StringBuffer

this class is not synchronized.

For particularly complex string-building needs, consider {ref java.util.Formatter}.

The majority of the modification methods on this class return

this

so that method calls can be chained together. For example:

new StringBuilder("a").append("b").append("c").toString()

.
