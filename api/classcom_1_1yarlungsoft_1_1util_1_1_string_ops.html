Splits a string into sections according to a separating character.

Parameters
stThe string to split into multiple substrings.
sepThe character to use as a separation point.
multipleTrue if we should combine multiple sep characters into a single separation token. If not, there will be blank strings between these characters.

If sep = ';' the string ';this;is;;a;;test;' will be split into:

multiple=true: "this", "is", "a", "test". multiple=false: "", "this", "is", "", "a", "", "test", "".
